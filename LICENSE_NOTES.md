# License Notes

This repository is a research and production-pattern archive.

## Core policy

Do not copy commercial game code or extracted commercial assets into this repository.

## Safe usage categories

### 1. Public permissive code

May be used if license allows it, but still record attribution and license.

Examples:

- MIT
- Apache-2.0
- BSD variants
- Unlicense / CC0 where clearly stated

### 2. Public copyleft code

Use carefully. It may affect the license of derivative work.

Examples:

- GPL
- LGPL
- AGPL

### 3. No clear license

Treat as all-rights-reserved. Do not copy code. Pattern reference only.

### 4. Fan games, mods, reverse-engineered projects

Often useful for structure and workflow learning, but risky for direct reuse.

Allowed:

- high-level architecture notes
- original rewritten pattern descriptions
- implementation ideas rewritten for our own project

Avoid:

- copied source code
- extracted commercial assets
- proprietary names, sprites, sounds, music

### 5. Commercial game references

Use only for design and production-pattern analysis. Do not preserve assets or source.

## Required fields in repo cards

Every repo analysis should include:

- license
- direct-use risk
- asset provenance risk
- code-copy status
- production pattern value

## Default stance

If license or provenance is unclear, mark as `REFERENCE_ONLY` or `CAUTION_LICENSE`.
